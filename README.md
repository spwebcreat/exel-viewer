# Excel Quick Viewer 📊

Excelファイルをアプリで開かなくても、中身をサクサク確認できる軽量ビューアアプリです。
クライアントとの電話中など、「今すぐ内容を確認したい」というシーンに最適です。

![アプリ画面](docs/images/screenshot.png)

## ✨ 特徴

- **高速プレビュー**: 重たいExcel本体を起動することなく、一瞬で中身を確認できます。
- **フォルダ管理機能**: よく使う見積もりフォルダなどを「追加」しておけば、次回からすぐにアクセスできます。
- **階層表示**: 登録した複数のフォルダ内のファイルを、ツリー形式で見やすく整理して表示します。
- **Excelで開く**: 編集が必要になったら、ボタンひとつでExcel本体を起動できます。
- **新旧形式対応**: `.xlsx` (現在の形式) だけでなく、古い `.xls` 形式にも対応しています。
- **検索機能**: ファイル内をキーワードで検索し、該当箇所をハイライト表示します。

## 📥 インストール方法

右側の [Releases](https://github.com/spwebcreat/exel-viewer/releases) ページから最新版をダウンロードしてください。

### Windowsの方
1. `Excel-Quick-Viewer_x.x.x_x64_en-US.msi` (または `.exe`) をダウンロードします。
2. ファイルをダブルクリックしてインストーラーを起動し、画面の指示に従ってインストールしてください。
   - ※ セキュリティ警告が出る場合がありますが、「詳細情報」→「実行」を選択して進めてください（署名未登録のため）。

### Macの方
1. `Excel Quick Viewer_x.x.x_aarch64.dmg` (M1/M2/M3 Mac用) または `_x64.dmg` (Intel Mac用) をダウンロードします。
2. ファイルを開き、アイコンをApplicationsフォルダにドラッグしてインストールしてください。

## 🚀 使い方

1. **アプリを起動**: `Excel Quick Viewer` アイコンをクリックして起動します。
2. **フォルダ追加**: 左上の「+ 追加」ボタンを押し、見積書などが入っているフォルダを登録します（設定は自動保存されます）。
3. **ファイル閲覧**: 左サイドバーのファイル一覧から見たいファイルをクリックします。
4. **Excel起動**: 編集が必要な場合は、右上の「Excelで開く」ボタンをクリックします。
5. **検索**: 右上の検索バーに文字を入力すると、該当するセルがハイライトされます。

## ⚠️ 「開けない」やセキュリティ警告が出た場合

本アプリは個人の開発用証明書を使用しているため、初回起動時にOSのセキュリティ警告が表示される場合があります。
ウイルスや破損ではありませんので、以下の手順で許可を与えてください。

### Macの場合
**「“Excel Quick Viewer.app”は壊れているため開けません」と表示される場合**

Appleのセキュリティ機能（Gatekeeper）が、インターネットからダウンロードされた未署名のアプリをブロックしています。以下の手順で解除できます。

1. アプリを `Applications` フォルダに移動してください。
2. ターミナルアプリを開きます（`Command + Space` で「Terminal」と検索）。
3. 以下のコマンドを入力して `Enter` を押します（パスワードを求められたら入力してください）。
   ```bash
   sudo xattr -cr "/Applications/Excel Quick Viewer.app"
   ```
4. これで通常通りクリックして起動できるようになります。

### Windowsの場合
**「WindowsによってPCが保護されました」と表示される場合**

1. 警告画面にある **「詳細情報」** というリンクをクリックします。
2. 画面下に現れる **「実行」** ボタンをクリックしてください。
